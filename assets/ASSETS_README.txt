Lairaboost App - Required Asset Files
======================================

Replace these placeholder files with actual Lairaboost branding:

1. icon.png (1024x1024) - App icon
   - Used for both iOS and Android
   - Should be the Lairaboost logo on dark bg (#1a1c24)

2. adaptive-icon.png (1024x1024) - Android adaptive icon foreground
   - Just the logo/icon portion (Android will apply its own mask)

3. splash.png (1284x2778) - Splash screen
   - Logo centered on dark background (#1a1c24)

4. notification-icon.png (96x96) - Android notification icon
   - Monochrome white icon
   - Used for push notification small icon on Android

5. favicon.png (48x48) - Web favicon (optional)
   - Only used if running in web mode

Theme colors:
- Background: #1a1c24
- Accent blue: #3B82F6
- Text: #ffffff
